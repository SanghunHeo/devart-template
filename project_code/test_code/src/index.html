This project is upgrade version of sandbox ocean 2011. Main goal of new version is to expend experience of sandbox ocean to world with chrome web apps.

“Sandbox Ocean” is a multi-user interactive installation. Users collaborate in creating a dynamic and animated world by changing and shaping the topography of a sandbox. The interface is simple, intuitive and playful. Customizing Microsoft’s Kinect system, our project adds
a new-media dimension to the playful sandbox of our childhood. “Sandbox Ocean” is an environment for collaborative creative play. The malleable sand interface fosters an activity that emphasizes mutual and collaborative play rather thvan competition. The physical interaction is intended to bring digital media back from the mediated world of the computer screen into the realm of tangible experience.
